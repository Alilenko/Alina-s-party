# Alina's Party 🎉

Мобільний сайт для святкування з завданнями, бінго, рейтингом та стрічкою подій.

## Технології

- **Next.js 15** — фронтенд
- **Supabase** — база даних, storage, realtime
- **Tailwind CSS** — стилі

## Supabase — чи вистачить безкоштовного тарифу?

**Так, більш ніж достатньо** для 10 гостей:

| Ліміт | Free tier | Наша вечірка |
|-------|-----------|--------------|
| Користувачі | 50 000 MAU | 10 |
| База даних | 500 MB | ~1 MB |
| Storage | 1 GB | ~50 MB фото |
| Realtime | ✅ | Стрічка + рейтинг |

## Налаштування Supabase

1. Створіть проєкт на [supabase.com](https://supabase.com)
2. У **SQL Editor** виконайте **один файл**: `supabase/setup-all.sql`
3. У **Settings → API** скопіюйте URL та anon key
4. Створіть `.env.local`:

```env
NEXT_PUBLIC_SUPABASE_URL=https://xxxxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJhbG...
```

## Запуск

```bash
npm install
npm run dev
```

Відкрийте [http://localhost:3000](http://localhost:3000)
